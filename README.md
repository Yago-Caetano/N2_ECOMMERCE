# N2_ECOMMERCE
Projeto da disciplina de Linguagem de Programação III
O objetivo é criar API's de um CRUD, nosso CRUD se baseia em uma aplicação de ecommerce
## Tecnologias
-Spring-boot
-Java 17
-Postman
-MySQL
